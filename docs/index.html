Bespin

Overview

We are documenting the Bespin code as beautifully as we can. For the front end JavaScript and Web code we are using the code illuminated tool created by Atul Varma.

Here is what we've got for documentation so far.

Bespin Front End Code

Core Files
  • Bespin Core
  • Editor Bootstrap Code
  • Core Editor Events and Bespin.Events Helper
Client Code
  • FileSystem: Remote File System
  • Server: Implementation of the Bespin Server API
  • Session Module: EditSession and SyncHelper
  • Settings Module: Settings core, and various stores
Command Line
  • CommandLine: The core command line module
  • Commands: The store that holds reused commands
  • Dashboard Commands
  • Editor Commands
Editor
  • Editor Actions: what the editor can do
  • Editor: The meat. The Core. paint()
  • Model: the document model the editor uses
  • Syntax Highlighter: shim over to whatever highlighter is used
  • Themes: Current theming
  • Toolbar: link external toolbar to the editor itself
  • Undo: Manage the undo / redo queue
Utilities
  • Clipboard: Handle the clipboard to copy and paste
  • Canvas: shim it up, have the standard work all over
  • Key: Key codes and simple handling
  • Navigate: Shoot people around the Bespin URLs (a.k.a. location.href wrapper)
  • Path: combine paths, get directories, and escape away
  • TextSelection: The ability to call $(elements).enableTextSelection to disable it
  • TokenObject: split up a string into pieces
  • URLBar will watch the hash location and fire an event when it changes
